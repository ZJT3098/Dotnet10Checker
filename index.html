.NET 10+ Runtime Checker

检测 .NET 10+ 运行时是否正确安装的 Windows 桌面工具

项目概览 文件结构 功能特性 架构设计 编译构建

项目概览

一个使用 C# + Windows Forms 构建的轻量级桌面工具，用于检测系统是否安装了 .NET 10 或更高版本的运行时。工具支持简体中文、繁体中文和英文三种语言界面。

文件结构

  • DotNet10Checker.csproj 项目配置文件
  • Program.cs 主程序源代码 (450行)

编译输出目录

目录 说明
bin/编译中间文件
obj/构建缓存
publish/x64 单文件发布
publish-arm64/ARM64 单文件发布

架构设计

代码模块

  • Program - 入口点，WinForms 应用启动
  • Language - 语言枚举 (zhCN/zhTW/en)
  • Strings - 多语言字符串资源管理
  • MainForm - 主窗口 UI 和业务逻辑
  • RuntimeInfo - 运行时信息数据模型
  • CheckResult - 检测结果数据模型

核心流程

  • 1. 查找系统 dotnet.exe
  • 2. 执行 dotnet --list-runtimes
  • 3. 解析输出获取运行时版本
  • 4. 判断是否存在 .NET 10+
  • 5. 更新 UI 显示结果

编译构建

项目配置 (csproj)

<PropertyGroup>
  <OutputType>WinExe</OutputType>
  <TargetFramework>net8.0-windows</TargetFramework>
  <PublishSingleFile>true</PublishSingleFile>
  <SelfContained>true</SelfContained>
  <SubsystemOutput>Windows</SubsystemOutput>
</PropertyGroup>

编译命令

# x64 版本 dotnet publish -c Release -r win-x64 -o publish

# ARM64 版本 dotnet publish -c Release -r win-arm64 -o publish-arm64
